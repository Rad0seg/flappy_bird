# Flappy Bird - Win32 API Implementation

Реализация классической игры Flappy Bird на **чистом C99** с использованием **Win32 API**. Проект демонстрирует навыки низкоуровневой разработки под Windows без использования сторонних движков или библиотек.

# Описание игры

**Flappy Bird** — аркадная игра, где игрок управляет птицей, пролетающей между трубами. Цель — набрать максимальное количество очков, избегая столкновений. Игра включает систему баффов/дебаффов, смену времени суток и прогрессивное увеличение сложности.

# Игровая механика

    Птица постоянно падает под действием гравитации
    Игрок заставляет птицу прыгать нажатием **SPACE**
    Столкновение с трубой или краем экрана заканчивает игру
    За каждую пройденную трубу начисляется **1 очко**
    Спавнятся баффы и дебаффы для усложнения/облегчения игры
    Каждые 10 очков меняется время суток (День → Закат → Ночь)
    Каждые 10 пройденных труб скорость растет, усложняя игру

# Управление

| Клавиша   | Действие                                       |
|-----------|------------------------------------------------|
| **SPACE** | Прыжок птицы / Начать игру                     |
| **P**     | Пауза (с обратным отсчетом 3-2-1 при возврате) |
| **R**     | Перезапуск после Game Over                     |
| **ESC**   | Выход из игры                                  |

# Система баффов и дебаффов

Вероятность спавна: **25% + 15% + 60%**

# Баффы 

| Символ | Название    | Эффект                                           | Длительность |
|--------|-------------|--------------------------------------------------|--------------|
| **S**  | Mini Bird   | Уменьшение птицы на 30%                          | 5 сек        |
| **C**  | Jump Points | **+1 очко за каждый прыжок**, все трубы исчезают | 5 сек        |

# Дебаффы 

| Символ | Название | Эффект                      | Длительность |
|--------|----------|-----------------------------|--------------|
| **H**  | Heavy    | Усиленная гравитация (×1.5) | 5 сек        |

# Система времени суток

Небо меняется каждые **10 очков**:

| Очки  | Время | Небо            | Облака      |
|------ |-------|-----------------|-------------|
|  0-9  | День  | Голубое         | Белые       |
| 10-19 | Закат | Оранжевое       | Розовые     |
| 20-29 | Ночь  | Темное + звезды | Серые       |
| 30+   | Цикл  | Повторяется     | Повторяется |

# Система прогрессивной сложности

Каждые **10 пройденных труб** сложность растет:

| Очки  | Скорость    | Расстояние | Уровень    |
|-------|-------------|------------|------------|
| 0-9   | 3.0         | 250px      |  Легко     |
| 10-19 | 3.5         | 253px      |  Нормально |
| 20-29 | 4.0         | 256px      |  Сложновато|
| 30-39 | 4.5         | 259px      |  Сложно    |
| 40-49 | 5.0         | 262px      |  Hard      |
| 50-59 | 5.5         | 265px      |  HARD!     |
| 60-69 | 6.0         | 268px      |  Very Hard |
| 70+   | 6.5+        | 271px+     |  Insane    |

**Промежуток между трубами всегда 180px.**

# Разделение на модули

Проект разделен на **два независимых модуля**:

# **logic/** — Игровая логика
**Файлы:** `src/logic/game_logic.h`, `src/logic/game_logic.c`

**Ответственность:**
- Хранение состояния игры (`GameState`)
- Физика птицы (гравитация, скорость, размер)
- Управление трубами (движение, спавн, переработка)
- Система коллизий (AABB)
- Спавн и управление баффами/дебаффами
- Подсчет очков и рекордов
- Система паузы с обратным отсчетом
- Смена времени суток
- Сохранение/загрузка рекордов в файл
